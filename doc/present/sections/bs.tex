\section{背景减 - Background Substraction}

\begin{frame}
    \frametitle{背景减技术简介}

    \begin{enumerate}
        \item 背景减(Background Substraction)是一种被广泛使用的动态目标检测技术。背景减技术通常需要从静止的相机中提取出``前景''
        \item 为了计算前景蒙版，在当前帧和背景之间进行减法运算，减去前几帧中习得的``背景''
    \end{enumerate}
